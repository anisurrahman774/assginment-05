Question-1: What are the different ways to select an element in the DOM?

I consider a DOM element "hard to be selected" if it fills 5 terms:
1.It doesn't have any identifiers whatsoever (ID's, Classes or unique attributes).
2.Its Xpath is created dynamically in each server creation of a webpage.
3.It's one of many elements of the same tag (say, div), and the number of these is also dynamic, so a specific item reference like document.querySelector('div'), will be principally ineffective.
4.It is a bit frustrating to target by the HTML alone given that the road from a parent element with an ID is quite large (like 25-30 elements). It's also impossible in case the ID is created dynamically.
5.Other elements share the same textContent of the element (though these will appear on viewport only if you'll open accordions that contain them).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.textContents: textContents is all text contained by an element and all its children that are for formatting purposes only.
2.innerText: innerText returns all text contained by an element and all its child elements.
3.innerHtml: innerHtml returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a JavaScript technique that uses a single event listener to handle events for multiple child elements. It's a design pattern that takes advantage of event propagation.

How does event delegation work?
.When an event occurs on a child element, it bubbles up the DOM tree.
.The parent element's event listener handles the event based on the target element.
.The parent element handles the event for all of its child elements.

Question-4: What is event bubbling in the DOM?

Bubbling in the DOM is an event propagation method. It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

To add, remove, and modify HTML elements using JavaScript, you can use methods like createElement , appendChild , removeChild , and properties like innerHTML and textContent . For example, to add an element, you can create it using document. createElement and then append it to a parent element using appendChild .
